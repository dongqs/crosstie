# Crosstie

Turn-Key Rails

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'crosstie'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install crosstie

## Usage

    $ crosstie new my_app
